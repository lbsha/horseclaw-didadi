# HorseClawDiDaDi 🐎

马力全开，爪到成功

## 简介

HorseClawDiDaDi 是一个普通的 Java Maven 项目，用于演示 HorseClaw 自动化开发。

## 技术栈

- Java 17
- JUnit 5
- Maven 3

## 快速开始

```bash
# 克隆
git clone https://github.com/lbsha/horseclawdidadi.git
cd horseclawdidadi

# 编译
mvn compile

# 测试
mvn test
```

## 项目结构

```
src/
├── main/java/com/horseclawdidadi/
│   └── App.java
└── test/java/com/horseclawdidadi/
    └── AppTest.java
```
